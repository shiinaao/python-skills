碾平列表是指把多层嵌套的列表中的元素归纳到一个列表中

```
li = [[1,2,3], [4,5], [6,7,8]]
# Out[10]: [1, 2, 3, 4, 5, 6, 7, 8]
```

常见方法不再记录, 只看花式解法

## itertools

```
li = [[1,2,3], [4,5], [6,7,8]]

import itertools

list(itertools.chain.from_iterable(li))
# Out[13]: [1, 2, 3, 4, 5, 6, 7, 8]

list(itertools.chain(*li))
# Out[38]: [1, 2, 3, 4, 5, 6, 7, 8]
```

## sum\(这个还是太魔幻了\)

```
li = [[1,2,3], [4,5], [6,7,8]]

sum(li, [])
Out[15]: [1, 2, 3, 4, 5, 6, 7, 8]
```
