Python 进制转换

### 二进制, 八进制, 十六进制 --> 十进制

```
In [5]: int('1000', 2)
Out[5]: 8
In [6]: int('12', 8)
Out[6]: 10
In [7]: int('1f', 16)
Out[7]: 31
```

### 十进制 --> 二进制, 八进制, 十六进制

```
In [10]: bin(8)
Out[10]: '0b1000'
In [11]: oct(10)
Out[11]: '0o12'
In [12]: hex(31)
Out[12]: '0x1f'
```

### 二进制 --> 十六进制

```
In [14]: hex(int('0101000', 2))
Out[14]: '0x28'
```

只有 int() 函数支持指定进制参数
